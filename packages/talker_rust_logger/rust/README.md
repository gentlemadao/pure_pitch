# talker_rust_logger (Rust Crate)

The Rust implementation for the `talker_rust_logger` Flutter package.

This crate provides:
1.  `init`: Initialize logging to Talker only.
2.  `init_with_logger`: Initialize logging to Talker AND an inner logger (e.g. platform logger).

## Usage

This crate is intended to be used with the `talker_rust_logger` Dart package.
Please refer to the [main documentation](../README.md) for setup instructions.
